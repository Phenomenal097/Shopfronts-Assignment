# Framework - Java Playwright

This is the Java Playwright automation framework for the SDET assignment.

## Tech Stack

- Java 17
- Playwright for Java
- TestNG
- Maven
- Jackson Databind

## Run

```bash
mvn test
```

## Main Packages

- `base` - browser and API request setup.
- `pages` - SauceDemo page objects.
- `tests/ui` - SauceDemo positive and negative UI tests.
- `tests/api` - FakeStore API CRUD, auth, data-driven and contract tests.
- `utils` - JSON contract validation helper.
